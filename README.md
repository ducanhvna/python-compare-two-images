# python-compare-two-images
 Python Compare Two Images

## Installation 

``` bash
conda create -n env python=3.6
pip install scikit-image
pip install matplotlib

```

## Linux
``` 
python compare.py
```

## MacOsX
```
pythonw compare.py
```
